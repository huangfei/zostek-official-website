## 4. 视觉设计方向

### 4.1 总体风格

参考用户提供的官网原型截图，整体应为：

- **浅色科技官网**：白色、冷白、淡蓝渐变背景。
- **蓝白银视觉系统**：ZosTek Blue 为主色，深海军蓝作为文字与强调色，电青色用于点缀，晶圆银与冷灰作为材料感。
- **B2B 高端半导体调性**：专业、克制、可信，不娱乐化，不潮玩化。
- **精密工程感**：网格、参数、毫米波弧线、晶圆纹理、芯片走线、圆角几何卡片。
- **轻量未来感**：可以有 3D 玻璃 Z、晶圆平台、芯片微光，但不要过度霓虹或游戏化。

### 4.2 色彩 Token

建议在 CSS / Tailwind theme 中定义以下色彩：

```css
:root {
  --zostek-blue: #0070FF;
  --zostek-blue-2: #0A6FFF;
  --deep-navy: #0A1A3A;
  --electric-cyan: #00E5FF;
  --wafer-silver: #C7CDD6;
  --graphite-gray: #4A4F57;
  --text-primary: #0A1A3A;
  --text-secondary: #526071;
  --text-muted: #7B8794;
  --line-soft: #DDE8F8;
  --bg-page: #F6FAFF;
  --bg-card: #FFFFFF;
  --clean-white: #FFFFFF;
}
```

### 4.3 字体建议

不需要引入字体文件。优先使用系统字体：

```css
font-family: Inter, "PingFang SC", "Microsoft YaHei", "Noto Sans SC", Arial, sans-serif;
```

标题：字重 `700-800`，深海军蓝。  
正文：字重 `400-500`，中性灰蓝。  
英文小标题：建议使用 uppercase、小字号、增加 letter-spacing。

### 4.4 页面尺寸与网格

- 目标桌面宽度：`1440px` 或 `1920px` 下均应良好展示。
- 内容最大宽度：`1280px`。
- 顶部导航高度：`72px` 左右。
- Section 垂直间距：`72px - 96px`。
- 卡片圆角：`16px - 24px`。
- 卡片边框：`1px solid #DDE8F8`。
- 阴影：轻量，不要厚重。
- 使用统一 12 栏或 24 栏网格。

### 4.5 视觉元素

可使用以下方式实现科技背景和图形，不依赖外部图片也可以完成：

- SVG / CSS 绘制毫米波同心弧线。
- SVG / CSS 绘制晶圆圆盘、芯片网格、蓝色微光。
- 右侧 Hero 使用一个大型蓝色玻璃质感 `Z`，可用 SVG、CSS 3D、或图片资源。
- 卡片内使用线性图标，图标风格统一：蓝色描边、圆角、简洁。
- 背景中加入浅蓝色电路走线、点阵、参数标注，如 `mmWave 60GHz+`、`Phase Array`、`System Solution`。
